# Report Workspace

Use this folder for the written research report, outline, drafts, figures, and final export notes.

The final report should be written from the structured tool records and evidence files, not from memory.

- [`tool_evaluations_draft.md`](tool_evaluations_draft.md) is the Day 24 prose draft for the five hands-on tool evaluation sections.
